# SheetsApp

## Setup
1. Update `index.html` → replace YOUR_GOOGLE_CLIENT_ID
2. Update `app.js` → set API to Apps Script URL
3. Host this folder anywhere (GitHub Pages, Netlify, Vercel, or local)
4. Open `index.html` in browser

## Features
- Table + Kanban views
- Add/Delete/Inline editing
- Offline caching with IndexedDB
- Row-level permissions
- Multi-sheet support
- Dark mode + animations
- Export to CSV / Excel support
